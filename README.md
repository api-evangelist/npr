# NPR (npr)

National Public Radio (NPR) APIs. The APIs support station finding, authentication, user management, and listening with audio recommendations tailored to a user's preferences.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/npr/refs/heads/main/apis.yml)

## APIs

### NPR Listening
Audio recommendations tailored to a user's preferences.

- **Base URL:** https://listening.api.npr.org/
- **Documentation:** https://dev.npr.org/guide/services/listening
- **Swagger:** https://listening.api.npr.org/v2/swagger.json
- **Tags:** Audio, Listening

### NPR Station Finder
NPR member station information lookup.

- **Base URL:** https://station.api.npr.org/
- **Documentation:** https://dev.npr.org/guide/services/station-finder
- **OpenAPI:** [openapi/npr-station-finder-openapi-original.yml](openapi/npr-station-finder-openapi-original.yml)
- **Tags:** Stations

### NPR Identity
User management API and entry point to user-specific information.

- **Base URL:** https://identity.api.npr.org/
- **Documentation:** https://dev.npr.org/guide/services/identity
- **OpenAPI:** [openapi/npr-identity-openapi-original.yml](openapi/npr-identity-openapi-original.yml)
- **Tags:** Identity, Users

### NPR Authorization
API authorization service.

- **Base URL:** https://authorization.api.npr.org/
- **Documentation:** https://dev.npr.org/guide/services/authorization
- **OpenAPI:** [openapi/npr-authorization-openapi-original.yml](openapi/npr-authorization-openapi-original.yml)
- **Tags:** Authorization
